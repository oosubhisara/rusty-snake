# snake-rust
Snake game written in Rust / macroquad

![Screenshot](snake-rust-screenshot.png)

## dependencies
- Rust (compiler)
- Libraries
  - macroquad

## Install Rust
https://rust-lang.org/tools/install/

## Build
```
./build.sh
```

## Run
```
cargo run
```

## Play

- Press *F1* to start single player game

- Press *F2* to start two player game
